Tom's Marvellous Cocktails

Technologies: MongoDB, Express.js, React, Node.js, Socket.io, Reactstrap

A MERN stack single page application that uses Socket.io to create a real-time order system.

On one device a user will select their drink, this order is then captured, added to a Mongo database and simultaneously synchronised to another device with the 'Order List' section displayed.

The user operating the 'order list' side of the application will receive the orders in real-time and can then delete items from the list as they are completed.

Being the most complex project I have worked on, it proved to be an insightful and challenging learning experience.
